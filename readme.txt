Description:
The Fotag Mobile app written in android.

Controls:
- the button besides the filter widget can clear the rating
- the image can be enlarged by touched, but not bigger than its origin size
- add image by url. When click the search button at the first time an EditText
will appear. Click search button again will add the image of the url in EditText 
field.


Development environment:
osx 10.13.4,  API 26: Android 8.0 (Oreo) w.Android Studio 3.1.3
